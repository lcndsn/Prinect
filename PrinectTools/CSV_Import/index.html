CSV Import Console

Preencha os dados de job e mídia. Use "+F" para adicionar sub-linhas do mesmo job (ex: Capa e Miolo separados). Ao criar uma sub-linha, os campos que pertencem só ao job-pai ou só ao arquivo ficam travados automaticamente na linha errada, pra evitar preenchimento duplicado ou no lugar errado.

Destino no servidor

Informe o nome do servidor do cliente para montar o caminho de rede da pasta de importação.

\\SRVPMC04\PTConfig\Connector\CSVImport

Depois de gerar o .csv, abra o Windows Explorer, cole este caminho na barra de endereço e mova o arquivo baixado para essa pasta.
