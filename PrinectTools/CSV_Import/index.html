Prinect · Connector Import

CSV Import Console

Preencha os dados de job e mídia. Use "+F" para adicionar sub-linhas do mesmo job (ex: Capa e Miolo separados). Campos que não se aplicam a cada tipo de linha ficam travados automaticamente.

Destino no servidor

Informe o nome do servidor do cliente para montar o caminho de rede da pasta de importação.

\\SRVPMC04\PTConfig\Connector\CSVImport

Depois de gerar o .csv, abra o Windows Explorer, cole este caminho na barra de endereço e mova o arquivo baixado para essa pasta. Esse nome de servidor também é usado no botão caminho ao lado dos campos URL de cada linha, pra montar o caminho da rede de arquivos automaticamente.
